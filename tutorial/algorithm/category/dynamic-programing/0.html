    动态规划

    将 dp 数组中的最值或者最后一个值定义为问题的解，该解可以通过以求得的子问题的解 + 状态转移函数求得

    根据公式直接推导

    1137. 第 N 个泰波那契数 (opens in a new tab)

    • dp[i] 的子问题： dp[i-1]/dp[i-2]/dp[i-3]
    • 状态转移函数：问题中的函数
    class Solution:
    def tribonacci(self, n: int) -> int:
        dp = list(range(38))
        dp[0] = 0
        dp[1] = 1
        dp[2] = 1
 
        for i in range(3,n+1):
            dp[i] = dp[i-1] + dp[i-2] + dp[i-3]
 
        return dp[n]

    1646. 获取生成数组中的最大值 (opens in a new tab)

    class Solution:
    def getMaximumGenerated(self, n: int) -> int:
        # 边界判定
        if n == 0:
            return 0
        if n == 1:
            return 1
 
        dp = [0] * (n+1)
        dp[0] = 0
        dp[1] = 1
        # 直接根据公式推
        for i in range(1,n+1):
            if i * 2 <= n:
                dp[2*i] = dp[i]
            if i * 2 + 1 <= n:
                dp[2*i+1] = dp[i] + dp[i+1]
 
        return max(dp) # 开销和在循环过程中进行是一样的 ON

    746. 使用最小花费爬楼梯 (opens in a new tab)

    • dp[i] 的子问题：dp[i-2]/dp[i-1]
    • 状态转移函数：爬两种台阶的最小值+当前爬楼梯的花费
    class Solution:
    def minCostClimbingStairs(self, cost: List[int]) -> int:
        # 题目范围 len(cost) >= 2，不需要边界判定
        dp = [0] * len(cost)
        dp[0] = cost[0]
        dp[1] = cost[1]
 
        for i in range(2,len(cost)):
            dp[i] = min(
                dp[i-2], # 爬两个楼梯
                dp[i-1], # 爬一个楼梯
            ) + cost[i]
 
        return min(dp[-1],dp[-2]) # 从倒数第一个或倒数第二个楼梯开始爬

    53. 最大子数组和 (opens in a new tab)

    • 子问题：nums[:i] 中的累积值
    • 状态转移函数 max_v = max(max_v, acc + nums[i])

    当遍历到 nums[i] 时，如果 sum(nums[:i]) 大于 0，那么 max_v = max(max_v, acc)：

    • 如果遍历到 i 时，i 之前的序列和 acc = sum(nums[:i]) < 0，那么之前的序列一定是可以被抛掉的（否则只会越加越小）
    • 如果遍历到 i 时，i 之前的序列和 acc = sum(nums[:i]) >= 0，那么最大值要么在遍历 i 之前的过程中产生，要么在 acc + nums[i] 中产生，所以 max_v = max(max_v, acc + nums[i])
    • 在第一个状态时 max_v = nums[0], acc = 0
    class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        acc = 0
        max_v = nums[0]
 
        for i in range(len(nums)):
            if acc < 0:
                acc = 0
 
            acc += nums[i]
            max_v = max(max_v, acc)
 
        return max_v

    338. 比特位计数 (opens in a new tab)

    # dp[i] 为 dp[i] 的1的个数
# dp[i>>1] = dp[i] - 1 #如果 i 是奇数
# dp[i>>1] = dp[i] #如果 i 是偶数
 
class Solution:
    def countBits(self, n: int) -> List[int]:
        dp = list(range(n+1))
 
        for i in range(n+1):
            dp[i] = dp[i>>1] + (i & 1)
 
        return dp
 
    def countBits1(self, n: int) -> List[int]:
        dp = list(range(n+1))
 
        for i in range(n+1):
            if i % 2 == 0:
                dp[i] = dp[i>>1]
            else:
                dp[i] = dp[i>>1] + 1
 
        return dp

    62. 不同路径 (opens in a new tab)

    第一行和第一列都是 1，右下角的路径数等于在上面一行下来和左边过来的路径数的和

    class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
 
        dp = [[0] * n for i in range(m)]
 
        for i in range(m):
            dp[i][0] = 1
 
        for j in range(n):
            dp[0][j] = 1
 
        for i in range(1,m):
            for j in range(1,n):
 
                dp[i][j] = dp[i-1][j] + dp[i][j-1]
 
        return dp[-1][-1]

    观察状态转移函数，dp[i-1][j] 来自上一次循环，dp[i][j-1] 来自同循环的前一个值，所以可以压缩二维数组为一维数组。

    class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [1] * n
        for i in range(1,m):
            for j in range(1,n):
                dp[j] = dp[j] + dp[j-1]
 
        return dp[-1]

    63. 不同路径 II (opens in a new tab)

    跟上题相同，唯一的区别是障碍物处的路径数归 0

    class Solution:
    def uniquePathsWithObstacles(self, obstacleGrid: List[List[int]]) -> int:
        m = len(obstacleGrid)
        n = len(obstacleGrid[0])
 
        dp = [1] * n
 
        for i in range(m):
            for j in range(n):
                if obstacleGrid[i][j] == 1:
                    dp[j] = 0
                elif i == 0 and j > 0: # 第一行判断
                    dp[j] = dp[j-1]
                elif j > 0: # 边界判断
                    dp[j] = dp[j] + dp[j-1]
 
        return dp[-1]

    121. 买卖股票的最佳时机 (opens in a new tab)

    重点是注意到 $max(|j-j'|)$ 通过拆项可以从 $O(n^2)$ 优化到 $O(n)$。类似的题都可以通过类似的方式求解。

    class Solution:
 
    def maxProfit(self, prices: List[int]) -> int:
        right = prices[:]
 
        for i in reversed(range(len(prices)-1)):
            right[i] = max(right[i],right[i+1])
 
        max_v = 0
        for i in range(len(prices)):
            max_v = max(right[i]-prices[i],max_v)
 
        return max_v

    随后发现空间复杂度可以优化到 O(1)

    class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        right = prices[-1]
        max_v = 0
        for i in reversed(range(len(prices))):
            right = max(right,prices[i])
            max_v = max(right-prices[i],max_v)
        return max_v

    64. 最小路径和 (opens in a new tab)

    dp[n][m]，先写状态转移公式：dp[i][j] = min(dp[i-1][j],dp[i][j-1]) + grid[i][j]，随后可以根据状态转移公式直接压缩空间复杂度。

    class Solution:
    def minPathSum(self, grid: List[List[int]]) -> int:
        # dp[i][j] = min(dp[i-1][j],dp[i][j-1]) + grid[i][j]
        n = len(grid)
        m = len(grid[0])
        dp = [0] * m
        for i in range(n):
            for j in (range(m)):
                if i == 0:
                    dp[j] = dp[j-1] + grid[i][j]
                else:
                    dp[j] = min(dp[j],dp[j-1]) + grid[i][j]
        return dp[-1]

    91. 解码方法 (opens in a new tab)

    类似于爬楼梯，但是要注意 '0'

    class Solution:
    def numDecodings(self, s: str) -> int:
        m = {str(i):i for i in range(1,27)}
 
        dp = [1] + [0] * len(s)
 
        for i in range(len(s)):
            if s[i] != '0':
                dp[i+1] += dp[i]
 
            if i>0 and s[i-1]+s[i] in m:
                dp[i+1] += dp[i-1]
 
        return dp[-1]

    上面的式子同一时间只出现三个状态，因此可以直接优化为 a,b,c 三个滚动变量

    class Solution:
    def numDecodings(self, s: str) -> int:
        m = {str(i):i for i in range(1,27)}
 
        a,b,c = 0,1,0 # dp[i-1],dp[i],dp[i+1]
        for i in range(len(s)):
            if s[i] != '0':
                c += b
 
            if i>0 and s[i-1]+s[i] in m:
                c += a
            a,b,c = b,c,0
 
        return b

    97. 交错字符串 (opens in a new tab)

    令 dp[i][j] 表示 s3[i+j-1] 是否存在交错字符串，则：

    • 如果 s3[i+j-1] == s2[j-1]，则 dp[i][j] = dp[i][j-1]
    • 如果 s3[i+j-1] == s1[i-1]，则 dp[i][j] = dp[i-1]
    class Solution:
    def isInterleave(self, s1: str, s2: str, s3: str) -> bool:
        if len(s1) + len(s2) != len(s3):
            return False
 
        if s1 == '':
            return s2 == s3
        if s2 == '':
            return s1 == s3
 
        n = len(s1)+1
        m = len(s2)+1
 
        dp = [[False] * m for i in range(n)]
        dp[0][0] = True
 
        for i in range(n):
            for j in range(m):
                if i == 0 and j == 0:
                    continue
 
                dp[i][j] = (
                    dp[i-1][j] and (s3[i+j-1] == s1[i-1]) or
                    dp[i][j-1] and (s3[i+j-1] == s2[j-1])
                )
 
        return dp[-1][-1]

    根据转移公式可得状态空间优化代码：

    class Solution:
    def isInterleave(self, s1: str, s2: str, s3: str) -> bool:
        if len(s1) + len(s2) != len(s3):
            return False
 
        if s1 == '':
            return s2 == s3
        if s2 == '':
            return s1 == s3
 
        n = len(s1)+1
        m = len(s2)+1
 
        dp = [False] * m
        dp[0] = True
 
        for i in range(n):
            for j in range(m):
                if i == 0 and j == 0:
                    continue
 
                dp[j] = (
                    dp[j] and (s3[i+j-1] == s1[i-1]) or
                    dp[j-1] and (s3[i+j-1] == s2[j-1])
                )
 
        return dp[-1]

    120. 三角形最小路径和 (opens in a new tab)

    简单分析可得状态转移函数为 dp[i][j] = min(dp[i-1][j],dp[i-1][j-1]) + triangle[i][j]

    class Solution:
    def minimumTotal(self, triangle: List[List[int]]) -> int:
        n = len(triangle)
        dp = [[float('inf')] * n for _ in range(n)]
        dp[0][0] = triangle[0][0]
 
        for i in range(1,n):
            for j in range(i+1):
                dp[i][j] = min(dp[i-1][j],dp[i-1][j-1]) + triangle[i][j]
 
        return min(dp[-1])

    从上到下数组越来越多，从下到上数组越来越少。因此从下到上选择路径其实更容易从转移函数上直接优化空间复杂度。此时状态转移函数是 dp[i][j] = min(dp[i+1][j],dp[i+1][j+1]) + triangle[i][j]。

    class Solution:
    def minimumTotal(self, triangle: List[List[int]]) -> int:
        n = len(triangle)
        dp = triangle[-1]
 
        for i in reversed(range(n-1)):
            for j in range(i+1):
                dp[j] = min(dp[j],dp[j+1]) + triangle[i][j]
 
        return dp[0]

    从上到下压缩空间复杂度时，三角形右对齐更容易压缩空间复杂度。
