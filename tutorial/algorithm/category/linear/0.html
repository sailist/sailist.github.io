    优先级队列

    优先级队列是一个完全二叉树，保证了所有的父节点均大于相应的子节点，此时根节点为所有元素的最大值。作为一个队列，相应的操作接口为 push/pop/top。push/pop 操作的复杂度均为 O(nlogn)，过程中核心算法是上升(lift_up)和下降(lift_down)。

    class PHeap:
    def __init__(self):
        self.lis = []
 
    def lift_up(self, ci, j):
        while ci > j:
            pi = (ci - 1) // 2
            if self.lis[ci][0] < self.lis[pi][0]:
                self.lis[ci], self.lis[pi] = self.lis[pi], self.lis[ci]
            else:
                break
            ci = pi
 
    def lift_down(self, i):
        while i < len(self.lis):
            ci = i * 2 + 1
            if ci >= len(self.lis):
                break
 
            if ci + 1 < len(self.lis) and self.lis[ci + 1][0] < self.lis[ci][0]:
                ci += 1
 
            if self.lis[i][0] > self.lis[ci][0]:
                self.lis[ci], self.lis[i] = self.lis[i], self.lis[ci]
            else:
                break
            i = ci
 
    def push(self, item):
        self.lis.append(item)
        self.lift_up(len(self.lis) - 1, 0)
 
    def pop(self, ):
        res = self.lis[0]
        tail = self.lis.pop()
 
        if len(self.lis):
            self.lis[0] = tail
            self.lift_down(0)
        return res
 
    def top(self):
        if len(self.lis):
            return self.lis[0]
        else:
            return None

    class Solution:
    def mergeKLists(self, lists: List[Optional[ListNode]]) -> Optional[ListNode]:
        # 全放在一起然后 sort， nlogn

        # 优先级队列 nlogk
 
        p = PHeap()
        for lis in lists:
            if lis:
                p.push((lis.val,lis))
 
        dummy = ListNode()
        head = dummy
        while len(p.lis):
            _,lis = p.pop()
            head.next = lis
            head = lis
            lis = lis.next
            if lis:
                p.push((lis.val,lis))
 
        return dummy.next

    862. 和至少为 K 的最短子数组 (opens in a new tab)

    前缀和 + 贪心 + 优先级队列。一遍循环，更新前缀和，同时将每个前缀和及其下标插入到小根堆中。每个更新前缀和时，从优先级队列中判断栈顶元素是否符合并更新最小 i-i'，注意当 i-i' 是更小时，后续的循环中 i-i' 一定更大，所以弹出 i' 是有效的剪枝，不会丢失状态空间。

    在队列顶部的元素是值更小，下标不一定距离当前 i 更近，因此需要通过 while 循环取到最后。

    该题还可以通过双端队列实现 $O(N)$ 的复杂度。

    class Solution:
    def shortestSubarray(self, nums: List[int], k: int) -> int:
        ans = 0
        queue = []
 
        # 最小堆
        heapq.heappush(queue,(0,-1))
        min_l = 1e6
        for i in range(len(nums)):
            ans += nums[i]
            tgt = ans - k
            while queue and tgt - queue[0][0] >= 0:
                min_l = min(min_l,i-heapq.heappop(queue)[1])
            heapq.heappush(queue,(ans,i))
 
        if min_l == 1e6:
            return -1
        return min_l

    解释一下单调双端队列的思想。该过程是不断的维护一个前缀和的单调递增（第二个 while 循环）的双端队列。同时在遍历到当前的前缀和时，从左到右尝试是否存在符合要求 curSum - preSumArr[q[0]] >= k 的队列元素，如果有则尝试更新最小区间（第一个 while 循环）并弹出左侧元素。

    不会丢掉有效状态空间的原因：

    • 第一个循环弹出左侧元素，哪怕之后的前缀和跟弹出的元素也符合要求 curSum - preSumArr[index] >= k ，区间也只可能更大，所以第一个循环的弹出一定是有效的
    • 第二个循环弹出右侧元素，因为在之前已经遍历过第一个循环，所以该元素的左侧不存在区间更小的值（存在也已经在之前循环更新完了）；而当前遍历的元素之后也会插入到队列中，且满足 preSumArr[q[-1]] >= curSum，即使在后续遍历过程中，存在一个元素 curSum 使得 curSum - preSumArr[q[-1]] >= k，也至少要比当前的元素的区间大，所以弹出符合条件的队列右侧元素也一定是有效的。
    class Solution:
    def shortestSubarray(self, nums: List[int], k: int) -> int:
        preSumArr = [0]
        res = len(nums) + 1
        for num in nums:
            preSumArr.append(preSumArr[-1] + num)
        q = deque()
        for i, curSum in enumerate(preSumArr):
            while q and curSum - preSumArr[q[0]] >= k:
                res = min(res, i - q.popleft())
            while q and preSumArr[q[-1]] >= curSum:
                q.pop()
            q.append(i)
        return res if res < len(nums) + 1 else -1

    2462. 雇佣 K 位工人的总代价 (opens in a new tab)

    class Solution:
    def totalCost(self, costs: List[int], k: int, candidates: int) -> int:
        ans, n = 0, len(costs)
        if candidates * 2 < n:
            pre = costs[:candidates]
            heapify(pre)
            suf = costs[-candidates:]
            heapify(suf)
            i, j = candidates, n - 1 - candidates
            while k and i <= j:
                if pre[0] <= suf[0]:
                    ans += heapreplace(pre, costs[i])
                    i += 1
                else:
                    ans += heapreplace(suf, costs[j])
                    j -= 1
                k -= 1
            costs = pre + suf
        costs.sort()
        return ans + sum(costs[:k])

    215. 数组中的第 K 个最大元素 (opens in a new tab)

    该题可以用快速选择、大根堆、小根堆来实现，其中，大根堆（O(nlogn)）比小根堆（O(nlogk)）的时间复杂度要大一些。

    大根堆实现

    class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        nums = [-i for i in nums]
        heapify(nums)
        for i in range(k):
            ans = heappop(nums)
        return -ans

    小根堆实现

    class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        res = []
        for i in range(len(nums)):
            if len(res) < k:
                heappush(res,nums[i])
            elif nums[i] > res[0]:
                heapreplace(res,nums[i])
 
        return res[0]

    快速选择虽然复杂度是 $O(n)$，但是分治的常数时间似乎过大，导致过不了。

    895. 最大频率栈 (opens in a new tab)

    python 自带的 heapq 维护的是小根堆，维护成大根堆需要内部数值取反

    class FreqStack:
    def __init__(self):
        self.cnt = Counter()
        self.heap = []
        self.index = 0
 
    def push(self, val: int) -> None:
        self.cnt[val] += 1
        heappush(self.heap, (-self.cnt[val],self.index,val))
        self.index -= 1
 
    def pop(self) -> int:
        cnt,ind,val = heappop(self.heap)
        self.cnt[val] -= 1
        return val

    6260. 矩阵查询可获得的最大分数 (opens in a new tab)

    class Solution:
    def maxPoints(self, grid: List[List[int]], queries: List[int]) -> List[int]:
        mem = Counter()
 
        n = len(grid)
        m = len(grid[0])
 
        visited = [[0] * m for i in range(n)]
 
        q = [(grid[0][0],0,0)]
 
        last = 0
        while q:
            cur, i, j = heappop(q)
            if visited[i][j] == 1:
                continue
 
            visited[i][j] = 1
            last = max(last, cur)
            mem[last] += 1
 
            for x, y in sorted([(x, y) for x, y in [
                (i + 1, j),
                (i - 1, j),
                (i, j + 1),
                (i, j - 1),
            ] if not (x < 0 or x >= n or y < 0 or y >= m)],
                                key=lambda x: grid[x[0]][x[1]]):
                heappush(q, (grid[x][y], x, y))
 
        sq = sorted(queries)
 
        sq_mem = {}
 
        ans = 0
        for k in sorted(mem.keys()):
            while sq and k >= sq[0]:
                sq_mem[sq.pop(0)] = ans
            ans += mem[k]
 
        while sq:
            sq_mem[sq.pop(0)] = ans
 
        return [sq_mem[k] for k in queries]
    class Solution:
    def maxPoints(self, G: List[List[int]], Q: List[int], dirs4 = ((-1, 0), (1, 0), (0, -1), (0, 1))) -> List[int]:
        m, n, q = len(G), len(G[0]), [(G[0][0], 0, 0)]
        ss, cur, res = [[False] * n for _ in range(m)], 0, [-1] * len(Q)
        ss[0][0] = True
        for idx, h in sorted(enumerate(Q), key=lambda t: t[1]):
            while q and q[0][0] < h:
                cur += 1
                _, x, y = heappop(q)
                for u, v in dirs4:
                    i, j = x + u, y + v
                    if 0 <= i < m and 0 <= j < n and not ss[i][j]:
                        ss[i][j] = True
                        heappush(q, (G[i][j], i, j))
            res[idx] = cur
        return res
